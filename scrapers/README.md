# Uso del scraper de Sodimac

El script `sodimac.py` permite limitar la cantidad de categorías a scrapear usando el flag `--max-categories`.

Ejemplo de uso:

```bash
python sodimac.py --max-categories 25
```

Esto limitará el scraping a las primeras 25 categorías encontradas.

Puedes ajustar el número según tus necesidades.
